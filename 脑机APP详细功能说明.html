详细功能说明

🏠 主界面功能

APP主界面包含以下5个核心功能入口：

  • 连接脑机 - 用于连接脑机设备并进行脑电波检测
  • 个人信息 - 查看和编辑用户个人资料
  • 心理地图 - 查看心理健康数据和趋势
  • 秘密花园 - 心理调节内容库，包含多种调节方式
  • 健康方案 - 健康商城页面，提供付费健康方案

🌸 秘密花园功能

用户点击秘密花园项目按钮后进入5个主题选项卡：

🎵 音乐
🎮 游戏
📖 有声书
📺 视频
🎨 涂鸦画布

提供各种类型的音乐列表，根据用户情绪和需求推荐相应音乐，支持播放、暂停、停止等功能。

导航：播放界面 → 回目录 → 回秘密花园主页

提供放松和调节情绪的游戏，帮助用户缓解压力和改善情绪状态。

导航：游戏界面 → 回目录 → 回秘密花园主页

包含各类有助于心理健康的有声书资源，用户可以根据需要选择收听。

导航：播放界面 → 回目录 → 回秘密花园主页

提供心理健康相关的视频内容，包括冥想指导、放松技巧等。

导航：播放界面 → 回目录 → 回秘密花园主页

数字涂鸦画布，让用户通过绘画表达情感，释放内心压力。

导航：绘画面板 → 保存作品 → 回目录 → 回秘密花园主页

🗺️ 心理地图功能

心理地图模块包含以下5个子功能：

  • 健康日历 - 以日历形式展示用户的情绪变化和健康状态
  • 心情日记 - 用户可以记录每日的心情和感受
  • 测试报告 - 显示最新的脑健康测试报告
  • 历史报告 - 查看过往的测试报告，便于对比分析
  • 健康曲线图 - 以图表形式展示用户健康趋势的变化

📊 脑健康测试功能

设备连接成功后用户可以进行脑健康测试（3分钟），完成后提示音并产生测试报告：

  • 正能量雷达图 - 可视化展示用户的积极情绪指标
  • 负能量雷达图 - 可视化展示用户的消极情绪指标
  • 六大指标数据 - 包括焦虑、抑郁、压力、疲劳、注意力、睡眠等维度
  • 差分红字提醒 - 对异常指标用红色字体突出显示
  • 综合结果解读与建议 - 提供专业的测试结果解释和个性化建议

🔗 智能关联调节功能

点击六大指标数据其中一项，系统会智能链接到秘密花园的5个主题选项卡，用户可以选择其中一个主题进行辅助调整，内容适配指标的性质：

抑郁
焦虑
压力
疲劳
注意力
睡眠

推荐调节方式：

导航路径：用户点击抑郁指标 → 进入秘密花园 → 点击音乐 → 修复抑郁情绪的音乐列表 → 点击播放、暂停、停止 → 回目录、回秘密花园主页
